2. 연산자

비교연산자

~= : 부정

인덱싱을 하려면
a(1)라고 하면 1번째 값 나옴
a(2)라고 하면 2번째 값 나온다. 참고로 세로로 읽은 후 가로로 읽는다.
a(3,2)라고 하면 됨. a행렬의 3행 2열의 값이 나옴.
a(2, :)하면 a행렬의 2행만 전부 뽑아냄
a(:, 4)하면 a행렬의 4열만 전부 뽑아냄

' : 전치행렬 만들기 : transpose .* = elementwise multiply = 아다마르 곱
